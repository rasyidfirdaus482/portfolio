Welcome to my new blog. I decided to build this portfolio using **Next.js App Router** and **CSS Modules** for a clean, scalable architecture. I wanted to maintain deep control over aesthetics while scoring 100 on Core Web Vitals.

## Why CSS Modules?

CSS modules isolate CSS by scoping class names locally by default. This solves the global namespace problem in CSS seamlessly.
